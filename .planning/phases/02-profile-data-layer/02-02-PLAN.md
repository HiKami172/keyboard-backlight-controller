  <behavior>
    ProfileManager manages a single JSON envelope file: {config_dir}/profiles.json.
    Envelope structure:
    {
      "last_profile": null | "name",
      "profiles": { "name": { "name": ..., "mode": ..., "r": ..., "g": ..., "b": ..., "speed": ... } }
    }

    Constructor: ProfileManager(config_dir: pathlib.Path | None = None)
    - Default: pathlib.Path.home() / ".config" / "kbd-backlight"
    - Calls config_dir.mkdir(parents=True, exist_ok=True) immediately
    - Stores config_dir / "profiles.json" as self._profiles_path

    _load() -> dict:
    - Reads profiles_path; returns envelope dict
    - On FileNotFoundError or JSONDecodeError: returns {"last_profile": None, "profiles": {}}
    - Never caches — always reads from disk

    _save(data: dict) -> None (ATOMIC):
    - tmp = self._profiles_path.with_suffix(".json.tmp")
    - tmp.write_text(json.dumps(data, indent=2, ensure_ascii=False), encoding="utf-8")
    - tmp.replace(self._profiles_path)  # rename(2) — atomic on Linux

    _dict_to_profile(raw: dict) -> Profile (static method):
    - fields = {f.name for f in dataclasses.fields(Profile)}
    - return Profile(**{k: v for k, v in raw.items() if k in fields})
    - Unknown keys are silently filtered — hand-edited JSON does not crash

    Public API (full interface for Phase 3/4):
    - list_profiles() -> list[str]
    - get_profile(name: str) -> Profile | None
    - get_all_profiles() -> dict[str, Profile]  # one _load() for N profiles
    - save_profile(profile: Profile) -> None  # create or overwrite by name
    - delete_profile(name: str) -> None  # no-op if not found; clears last_profile if matched
    - rename_profile(old_name: str, new_name: str) -> None
      → raises KeyError if old_name not found
      → raises ValueError if new_name already exists
      → updates last_profile if it matched old_name
    - get_last_profile() -> Profile | None
    - set_last_profile(name: str) -> None

    TDD cases → expected outcomes:
    - save_profile(p) then new ProfileManager(same dir).get_profile(p.name) → p (persistence survives restart)
    - list_profiles() on empty manager → []
    - save_profile × 2 then list_profiles() → [name1, name2]
    - delete_profile("Work") when last_profile=="Work" → last_profile becomes None in JSON
    - rename_profile("Work", "Office") when last_profile=="Work" → last_profile becomes "Office"
    - rename_profile("NoExist", "X") → raises KeyError
    - rename_profile("Work", "Gaming") when "Gaming" already exists → raises ValueError
    - _load() on file with unknown key "notes" in a profile entry → loads without error, unknown key ignored
    - get_last_profile() when last_profile null → None
    - get_last_profile() when last_profile is valid name → returns Profile
    - get_all_profiles() returns dict[str, Profile] with all saved profiles
    - save_profile(p) overwrites if same name already exists (no error)
    - delete_profile("NoExist") → no exception (no-op)
  </behavior>
