  <behavior>
    Profile is a @dataclass with fields: name (str), mode (str), r (int), g (int), b (int), speed (int).

    __post_init__ validation rules (same as BacklightController — intentional duplication for hardware-independence):
    - name: non-empty, non-whitespace-only → raise ValueError("Profile name cannot be empty")
    - mode: must be in {"static", "breathing", "color_cycle", "strobe"} → raise ValueError
    - r, g, b: each 0-255 inclusive → raise ValueError
    - speed: must be 0, 1, or 2 → raise ValueError

    VALID_MODES: ClassVar[set[str]] = {"static", "breathing", "color_cycle", "strobe"}
    ClassVar fields are excluded from dataclasses.asdict() — verified Python 3.12.3.

    ProfileError(ValueError): custom error class for profile-specific validation errors.
    NOTE: Research recommends using ValueError directly in __post_init__. Use ProfileError as
    a ValueError subclass so callers can catch either. Raise ProfileError (not plain ValueError)
    from __post_init__ — this satisfies both catch styles.

    Serialization: dataclasses.asdict(profile) → dict with exactly 6 keys.
    Roundtrip: Profile(**dataclasses.asdict(p)) == p must hold.

    TDD cases → expected output:
    - Profile("Work", "static", 255, 128, 0, 0) → no exception; asdict has 6 keys
    - Profile("", "static", 255, 0, 0, 0) → raises ProfileError
    - Profile("  ", "static", 255, 0, 0, 0) → raises ProfileError
    - Profile("X", "disco", 255, 0, 0, 0) → raises ProfileError (unknown mode)
    - Profile("X", "static", 256, 0, 0, 0) → raises ProfileError (r out of range)
    - Profile("X", "static", 0, -1, 0, 0) → raises ProfileError (g out of range)
    - Profile("X", "static", 0, 0, 0, 3) → raises ProfileError (speed invalid)
    - Profile("X", "color_cycle", 0, 0, 0, 2) → no exception (valid)
    - Profile("X", "static", 0, 0, 0, 0) roundtrip: Profile(**asdict(p)) == p → True
    - VALID_MODES ClassVar not in asdict output → len(asdict(p)) == 6
  </behavior>

    2. Create kbd_backlight/profiles/profile.py:
       ```python
       import dataclasses
       from typing import ClassVar

       class ProfileError(ValueError):
           """Raised when Profile data is invalid."""

       @dataclasses.dataclass
       class Profile:
           VALID_MODES: ClassVar[set[str]] = {"static", "breathing", "color_cycle", "strobe"}

           name: str
           mode: str
           r: int
           g: int
           b: int
           speed: int

           def __post_init__(self) -> None:
               if not self.name or not self.name.strip():
                   raise ProfileError("Profile name cannot be empty")
               if self.mode not in self.VALID_MODES:
                   raise ProfileError(
                       f"Unknown mode '{self.mode}'. Valid: {sorted(self.VALID_MODES)}"
                   )
               if not all(0 <= c <= 255 for c in (self.r, self.g, self.b)):
                   raise ProfileError(
                       f"RGB values must be 0-255, got ({self.r}, {self.g}, {self.b})"
                   )
               if self.speed not in (0, 1, 2):
                   raise ProfileError(f"Speed must be 0, 1, or 2, got {self.speed}")
       ```
